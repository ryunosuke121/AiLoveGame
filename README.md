# AIしAIされGPT
## 概要
ハックツハッカソン ジンベエカップ（5/21,22)にて開発した、ChatGPTを活用した恋愛シミュレーションゲーム
好みの異性について質問に答えることで、ChatGPTが成り切ってくれます。
openAIの画像生成APIを使って生成した画像をremBgライブラリを使って、背景を透過させています。

## 使用技術
### フロントエンド
Nextjs, React, tailwindcss
### バックエンド
fastAPI, ChatGPT-API, MySQL
### デプロイ環境
GCP-CloudRun Vercel
